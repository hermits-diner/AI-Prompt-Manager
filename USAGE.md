# AI 프롬프트 매니저 사용법

고등학교 교사를 위한 프롬프트와 AI 서비스 링크를 한곳에서 관리하는 앱입니다.

---

## 화면 구성

- **왼쪽 (약 75%)**: 프롬프트 목록과 검색·필터
- **가운데 구분선**: 드래그해서 좌우 비율 조절 가능
- **오른쪽 (약 25%)**: AI 링크(채팅·작문, 이미지, 음악·음성, 도구·기타) + **빠른 메모**

---

## 1. 프롬프트 관리

### 새 프롬프트 추가

1. 상단 **「새로 추가」** 버튼 클릭
2. **제목** · **프롬프트 내용** 입력 (필수)
3. **카테고리** 선택 (드롭다운)
4. **태그**는 선택 사항. 입력 후 **Enter** 또는 **+** 버튼으로 추가
5. **저장하기** 클릭

### 프롬프트 수정

- 카드에서 **제목 또는 내용 영역**을 클릭하면 **상세 보기** 창이 열립니다.
- 상세 창에서 **「수정」** 버튼을 누르면 수정 폼이 열립니다.
- 또는 카드 오른쪽 **⋮(더보기)** → **「수정」** 선택해도 됩니다.

### 프롬프트 삭제

- 카드 **⋮(더보기)** → **「삭제」** → 확인 후 삭제됩니다.

### 프롬프트 복제

- **⋮(더보기)** → **「복제」** 를 누르면 같은 내용의 새 프롬프트가 만들어집니다. 제목 끝에 「(복사)」가 붙습니다.

### 즐겨찾기

- 카드 오른쪽 **별 아이콘**을 클릭하면 즐겨찾기에 추가/해제됩니다.
- 즐겨찾기한 항목은 **「프롬프트 즐겨찾기」** 섹션에 모여서 보입니다. 여기서 **복사** 버튼으로 내용을 복사하거나, **별**을 눌러 해제할 수 있습니다.

### 내용 복사

- 카드 하단 **「복사」** 버튼 또는 상세 보기 창의 **「클립보드에 복사」** 버튼을 누르면 프롬프트 **내용 전체**가 클립보드에 복사됩니다.  
  → AI 채팅 창에 **Ctrl+V**로 붙여넣기 하면 됩니다.

---

## 2. 검색·필터·정렬

- **검색창**: 제목, 내용, 태그에 포함된 단어로 검색합니다.
- **필터** 버튼(슬라이더 아이콘)을 누르면:
  - **카테고리**: 특정 카테고리만 보기 (여러 개 선택 가능)
  - **즐겨찾기만**: 즐겨찾기한 프롬프트만 보기
  - **정렬**: 최신순 / 오래된순 / 이름순
- **필터 초기화**: 적용된 필터가 있으면 **「초기화」** 버튼이 나타납니다. 클릭하면 검색어·카테고리·즐겨찾기·정렬이 모두 기본값으로 돌아갑니다.

---

## 3. 가져오기·내보내기

### 내보내기 (백업)

- 상단 **「내보내기」** 클릭 → 현재 **프롬프트 + 카테고리**가 JSON 파일로 다운로드됩니다.  
  다른 PC나 브라우저에서 쓰려면 이 파일을 보관해 두세요.

### 가져오기 (일괄 추가)

- **「가져오기」** 클릭 후:
  - **JSON**: `[{"title":"제목","content":"내용","category":"카테고리명","tags":["태그1"]}]` 형식의 배열
  - **CSV**: `title,content,category,tags` 헤더 + 한 줄에 한 프롬프트 (태그는 `|`로 구분, 예: `"태그1|태그2"`)
- 텍스트를 붙여넣고 **가져오기**를 누르면 목록에 일괄 추가됩니다. 기존 데이터는 유지됩니다.

---

## 4. 카테고리 관리

- 상단 **「카테고리」** 버튼 클릭 → **카테고리 관리** 창이 열립니다.
- **추가**: 새 카테고리 이름 입력 후 추가
- **수정** / **삭제**: 각 카테고리 옆 버튼으로 수정·삭제
- **기본 카테고리로 초기화**: 고등학교 교사용 기본 5개 카테고리(수업·강의, 평가·채점, 학습지도·상담, 학급·행정, 자료·연구)로 되돌립니다.  
  ⚠️ 기존에 만든 카테고리는 삭제되고, 해당 카테고리를 쓰던 프롬프트는 **「미분류」**로 바뀝니다.

---

## 5. AI 링크 (오른쪽 패널)

- **채팅·작문**, **이미지**, **음악·음성**, **도구·기타** 등 카테고리별로 AI 서비스 링크가 나열됩니다.
- **링크 이름**을 클릭하면 해당 서비스가 **새 브라우저 탭**에서 열립니다.  
  → 미리 프롬프트를 **복사**해 두었다가 새 탭에서 **Ctrl+V**로 붙여넣어 사용하면 됩니다.
- **「관리」** 버튼을 누르면:
  - AI 링크 **카테고리** 추가·수정·삭제
  - 각 카테고리 안에 **링크** 추가·수정·삭제 (이름, URL)
  - **기본값으로 초기화**: 채팅·작문, 이미지, 음악·음성, 도구·기타 등 기본 목록으로 되돌리기

---

## 6. 빠른 메모 (오른쪽 하단)

- AI에 붙여넣기 전에 **문장만 살짝 고치거나 메모**할 때 사용하는 메모장입니다.
- 입력한 내용은 **자동으로 저장**되며, 새로고침해도 유지됩니다(같은 브라우저 기준).
- **복사** 버튼: 메모 전체를 클립보드에 복사  
- **지우기** 버튼: 메모 내용 전체 삭제

---

## 7. 데이터 저장 방식

- 모든 데이터(프롬프트, 카테고리, AI 링크, 빠른 메모)는 **브라우저 localStorage**에만 저장됩니다.
- **다른 기기·다른 브라우저**에서는 보이지 않습니다.
- **시크릿 모드**나 **캐시/데이터 삭제**를 하면 해당 기기에서만 데이터가 사라질 수 있으므로, 중요한 프롬프트는 **「내보내기」**로 JSON 백업을 해 두는 것이 좋습니다.

## 요약

| 하고 싶은 일           | 어떻게 하면 되나요                                      |
|------------------------|--------------------------------------------------------|
| 새 프롬프트 추가        | **새로 추가** → 제목·내용 입력 → 저장                  |
| 프롬프트 수정          | 카드 클릭 → 상세 보기 → **수정** / 또는 ⋮ → 수정      |
| 프롬프트 복사해서 사용 | 카드 **복사** 또는 즐겨찾기에서 **복사** → AI 탭에서 Ctrl+V |
| 즐겨찾기 모아보기      | 별 아이콘으로 즐겨찾기 → **프롬프트 즐겨찾기** 섹션에서 확인 |
| 검색·필터              | 검색창 + **필터** 버튼(카테고리, 즐겨찾기만, 정렬)     |
| 백업                   | **내보내기** → JSON 저장                               |
| 다른 PC에서 쓰기       | **가져오기**로 내보낸 JSON 붙여넣기                     |
| AI 사이트 열기          | 오른쪽 **AI 링크**에서 원하는 서비스 클릭 → 새 탭에서 사용 |
| 잠깐 메모              | 오른쪽 하단 **빠른 메모**에 입력 → 필요 시 복사       |

이 문서는 프로젝트 루트의 **USAGE.md** 파일로도 저장되어 있습니다.
